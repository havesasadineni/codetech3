# ChatApplication

A simple multi-threaded chat application built in Java that allows multiple clients to connect to a server and exchange messages in real time.

## 🚀 Features

- Server-client architecture using sockets
- Multithreaded server to handle multiple clients
- Simple text-based UI
- Real-time messaging

## 🛠️ Technologies Used

- Java (JDK 17)
- Java Sockets
- Swing (for GUI, if used)
